# Папка assets

Скопируйте сюда файл модели TensorFlow Lite после конвертации.

**Инструкция:**

1. Запустите скрипт `convert_keras_to_tflite.py` из корня проекта
2. Скрипт автоматически создаст файл `waste_classifier.tflite` в этой папке
3. Или вручную скопируйте конвертированную модель сюда

**Файл модели должен называться:** `waste_classifier.tflite`
